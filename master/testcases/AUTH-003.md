---
id: AUTH-003
title: アカウントロックアウト（連続失敗）
requirement: REQ-LOGIN-001
priority: high
category: auth
type: security
preconditions:
  - ユーザーアカウントが存在する
  - アカウントが有効
  - ロックアウト閾値が5回に設定されている
tags:
  - security
  - login
---

# 手順

1. ログイン画面を開く
2. 有効なユーザーIDと誤ったパスワードで5回連続ログインを試みる
3. 6回目のログイン試行を行う

# 期待結果

- 5回失敗後にアカウントがロックされる
- 「アカウントがロックされています」旨のメッセージが表示される
- ロックされた状態では正しい認証情報でもログインできない

# 手順

1. ヘッダーのユーザーアイコンをクリックし「プロフィール編集」を選択する
2. 氏名フィールドを新しい値（例: "山田 花子"）に変更する
3. メールアドレスを有効な形式（例: "hanako@example.com"）に変更する
4. 「保存」ボタンを押下する

# 期待結果

- 「プロフィールを更新しました」成功メッセージが表示される
- 変更した氏名・メールアドレスが画面に反映される
- DBに更新日時が記録される
